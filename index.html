Pixel Art Camera

Pixel Art Camera is a simple tool for Unity that can be used to achieve pixel perfect graphics at any resolution.

It works by rendering the game at the target resolution and then upscale it to the biggest possible integer factor.
This way, the pixel art will always remain pixel perfect with no artifacts.

Pixel Art Camera is open source and is avaiable for free on the Unity Asset Store.

BoxBoxBox

BoxBoxBox is a work-in-progress motorsport management game that focuses on emergent-narrative.

The game features an accurate physics simulation of how real vehicles works, while the driving AI is based on context steering behaviours and can be fine-tuned in order to simulate different types of drivers and racing styles.

Thanks to its smart AI and a rich cast of drivers with different skills and abilities, the game can endlessly generate engaging and unpredictable races.

Disboard

Disboard is a C++ framework useful for studying and developing AIs for turn based games.

It includes a basic game engine that can be expanded to implement any type of sequential turn based game. It also features different AIs that can play any game made with Disboard.

Disboard is currently being ported to Unreal Engine.

This projects benefits from the HPC (High Performance Computing) facility of the University of Parma, Italy.

OVERHEAT

OVERHEAT was my entry for the Blackthornprod GAME JAM #3, a one-week Game Jam with over one thousand entries.

The theme of the jam was 'Less is More', so the game gets more difficult by giving you less power ups every time you beat it.

The game finished in the top 20% of the entries despite being made only in the last three days of the jam.

LoR Wiki

LoR Wiki was the first companion app for Legends of Runeterra to land on Android. It has over 5k downloads and a 4+ stars rating.
